# Heartify

Heartify returns hearts back to Spotify!  
This extenstion replaces Spotify's new "Add to Collection" buttons with the good old heart buttons. As they should, hearts indicate whether a track is in your **Liked Songs**, not just if it's in any of your playlists.

## Install

Heartify is available for both Chrome and Firefox. Click one of the badges below to install Heartify on your browser:
